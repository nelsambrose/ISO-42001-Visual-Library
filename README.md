## Folder convention

```text
cards/
  reference/
  funny/
  professional/
```

Use the same topic filename across folders where applicable. For example, an `ai-risk` topic should use `cards/reference/ai-risk.md`, `cards/funny/ai-risk.png`, and `cards/professional/ai-risk.png`.

When image cards are added, this README should embed them directly so GitHub displays the visual library on the repository landing page.
